BlockBoy Kit

Algemene Vragen

Wat is dit project?

+

Dit is een drop-in elektronica kit die de officiële LEGO Game Boy set (#72046) volledig functioneel maakt. De kit transformeert het LEGO displaystuk in een écht werkende, draagbare Game Boy die echte Game Boy en Game Boy Color spellen kan spelen!

Heb ik de LEGO Game Boy set nodig?

+

Ja, deze kit is specifiek ontworpen als upgrade voor de officiële LEGO Nintendo Game Boy set (#72046). De LEGO set is essentieel en wordt niet meegeleverd.

Technische Specificaties

Wat zijn de specificaties?

+
ProcessorESP32-S3 microcontroller
EmulatorRetro-Go (Game Boy, GBC, Game & Watch, NES, Sega, DOOM)
Display2,8" IPS-scherm met achtergrondverlichting (ILI9341)
BatterijOplaadbare lithiumbatterij, ~1,5 uur speeltijd
OpladenUSB-C poort
OpslagMicroSD-kaart (niet inbegrepen)
AudioIngebouwde speaker met fysieke volumeknop
HelderheidInstelbaar met fysieke knoppen
BesturingVolledig functionele knoppen en D-pad

Welke games kan de kit spelen?

+

De kit kan Game Boy (DMG) en Game Boy Color (GBC) games spelen via ROM-bestanden die zijn opgeslagen op een microSD-kaart.

Ondersteunt de kit echte Game Boy cartridges?

+

Nee. Deze versie maakt gebruik van emulatie met ROM-bestanden op een SD-kaart. Dit houdt de kit simpel en behoudt de integriteit van vintage consoles.

Kan de kit link kabel multiplayer?

+

Nee, de Link kabel-functionaliteit wordt niet ondersteund.

Hoe lang gaat de batterij mee?

+

Ongeveer 1,5 uur gameplay. De batterijduur kan variëren afhankelijk van de ingestelde schermhelderheid en het gebruik.

Hoe laad ik de kit op?

+

Opladen gaat via de USB-C poort die zich boven het volumewiel bevindt. Een standaard USB-C kabel is alles wat u nodig heeft.

Installatie & Montage

Is de installatie moeilijk?

+

Nee, de kit is ontworpen als een drop-in oplossing. Er is geen solderen nodig. De installatie vereist wel kleine interne aanpassingen aan het LEGO-model, zoals het verplaatsen van enkele bestaande bouwstenen, om ruimte te maken voor de elektronica.

Als u LEGO kunt bouwen, kunt u deze kit installeren.

Wat zit er in de kit?

+

De kit bevat alle elektronische onderdelen en aansluitingen die u nodig heeft om uw LEGO-model om te bouwen:

Hoofdprintplaat (PCB) met daarop:

  • ESP32-S3 microcontroller
  • 2,8" IPS-display
  • Oplaadbare lithiumbatterij
  • Aan/uit-schakelaar
  • Fysieke knoppen voor helderheids- en volumeregeling

Besturingsprintplaten (PCBs):

  • D-pad PCB: Voor de richtingsknoppen
  • A/B PCB: Voor de A- en B-actieknoppen
  • Select/Start PCB: Voor de Select- en Start-knoppen

Aansluitingen:

  • Verschillende flatcables (flexibele lintkabels) voor plug-and-play verbindingen

Keuze bij bestelling: Voor de D-pad en A/B-knoppen kunt u een keuze maken tussen tastbare klikknoppen (duidelijke 'klik') of membraanknoppen (zachter, traditioneel Game Boy gevoel).

Niet inbegrepen:

  • LEGO Game Boy set (#72046)
  • MicroSD-kaart
  • Gamebestanden (ROMs)
  • USB-C oplaadkabel

Heb ik gereedschap nodig?

+

Nee, er is geen speciaal gereedschap vereist. Alles is plug-and-play. U heeft alleen de LEGO-handleiding en de instructies van mijn kit nodig.

Hoe lang duurt de installatie?

+

De meeste gebruikers voltooien de installatie in minder dan 60 minuten.

Juridisch & Ethiek

Belangrijke Aansprakelijkheidsverklaring (Disclaimer)

Mijn kit is een onafhankelijk product. Dit project is op geen enkele wijze verbonden, geassocieerd, geautoriseerd, onderschreven door of officieel verbonden met de LEGO Groep of Nintendo Co., Ltd. De handelsmerken "LEGO", "Nintendo", "Game Boy" en alle gerelateerde namen, merken en afbeeldingen zijn eigendom van hun respectievelijke houders.

Is dit legaal?

+

Ja. De hardware kit zelf is volledig legaal. Echter:

  • Emulators zijn legaal.
  • U moet de originele spellen bezitten voor de ROM-bestanden die u gebruikt.
  • Ik lever, verkoop of link niet naar auteursrechtelijk beschermde gamebestanden.
  • Dit product is uitsluitend bedoeld voor persoonlijk gebruik.

Verkoop ik ROMs?

+

Nee. Ik lever, verkoop of distribueer géén auteursrechtelijk beschermde gamebestanden. Gebruikers zijn zelf verantwoordelijk voor het legaal verkrijgen van hun spellen.

Is dit open source?

+

De Retro-Go software die de kit gebruikt is open source. De hardware/PCB ontwerpen van mijn kit zijn momenteel niet open source. Ik overweeg om designbestanden in de toekomst vrij te geven.
