# BlockBoy - Werkende LEGO Game Boy Mod

**Transformeer je LEGO Game Boy (#72046) in een echte, werkende handheld console!**

BlockBoy is een drop-in elektronica kit die de officiÃ«le LEGO Nintendo Game Boy set volledig functioneel maakt. Speel Game Boy en Game Boy Color games op je eigen LEGO Game Boy.

## Specificaties

- **Microcontroller:** ESP32-S3
- **Display:** 2.8" IPS TFT
- **Software:** [Retro-Go](https://github.com/ducalex/retro-go) emulator
- **Batterij:** 300mAh oplaadbaar (~1.5 uur speeltijd)
- **Opladen:** USB-C
- **Installatie:** Geen solderen nodig - plug and play
